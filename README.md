### Requirements
###### PHP ^8.1
###### Node.js & npm
###### Sqlite for local
###### Heroku CLI

### Installing with Git:

В вашем терминале запустите:

  ```sh
   git clone https://github.com/qwelp/php-project-lvl3
  ```
  ```sh
 cd php-project-lvl3
  ```
  ```sh
 make setup
  ```
  ```sh
 make start
  ```
