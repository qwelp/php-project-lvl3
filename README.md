### Installing with Git:

В вашем терминале запустите:

  ```sh
   git clone https://github.com/qwelp/php-project-lvl3
  ```
  ```sh
 cd php-project-lvl3
  ```
  ```sh
 make setup
  ```
  ```sh
 make start
  ```
